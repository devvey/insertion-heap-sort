      СРАВНИТЕЛЬНЫЙ АНАЛИЗ АЛГОРИТМОВ СОРТИРОВКИ
           Insertion Sort vs Heap Sort

Курсовая работа по дисциплине "Алгоритмы и структуры данных"

Автор: Заболотский Богдан Александрович
Группа: Б24-191-2
Дата: 19.12.2025

Данный проект представляет собой сравнительный анализ двух 
алгоритмов сортировки:
- Сортировка вставками (Insertion Sort)
- Пирамидальная сортировка (Heap Sort)

Цель работы: 
- Теоретическое исследование вычислительной сложности алгоритмов
- Практическая реализация на языке C++
- Экспериментальное тестирование производительности
- Сравнение с библиотечной реализацией std::sort

СБОРКА:

Windows (через командную строку):
1. Откройте командную строку (cmd)
2. Перейдите в папку проекта:
   cd C:\путь\к\проекту\coursework_sorting
3. Создайте папку build и перейдите в нее:
   mkdir build && cd build
4. Сгенерируйте проект CMake:
   cmake ..
5. Соберите проект:
   cmake --build . --config Release

Visual Studio (Windows):
1. Откройте папку проекта в Visual Studio 2022
2. Выберите конфигурацию "Release"
3. Нажмите "Собрать" → "Собрать решение"

ЗАПУСК:

После сборки исполняемый файл будет находиться в папке build/

Windows:
1. Откройте командную строку
2. Перейдите в папку build/Release/
3. Запустите программу:
   coursework_sorting.exe
